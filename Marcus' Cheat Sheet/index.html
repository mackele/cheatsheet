Cheat Sheet Useful References

HTML Table

Tag Name Description
<Table> Table Defines an HTML table
<tr> Table row Defines and creates a new row
<th> Table header Creates header with bold and centered text
<td> Table data Regular and left-aligned
border Border Adds border to table, example 1px solid black
border-collapse Border collapse Collapses the border into one solid table, example border-collapse: collapse;
padding Padding Specifies space between the content and borders, example padding: 15px;

CSS Selectors

Selector Example Description
.class .yesPlease Selects all elements with yesPlease class
#id #yesPlease Selects element with id="yesPlease"
* * Selects all elements
element p Selects all <p> elements
element.class p.yesPlease Selects all <p> elements within yesPlease class
element, element p, h2 Selects all <p> and all <h2> elements
element element div p Selects all <p> elements within <div> elements
